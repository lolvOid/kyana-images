## Key Visual Elements Across All Images

### Colors Available
- **Black** (default) - Matte black silicone, glossy black display
- **White** (future variant)
- **Silver** (future variant)
- **Space Grey** (future variant)

### Core Components Always Visible
1. **Core Device**: Square 33mm case, glossy display, speaker left, button right
2. **Ring Frame**: Square frame, 4 copper magnetic connectors inside, 6 copper pins outside top
3. **Silicon Strap**: 6 rectangular sensor windows with green LED indicators
4. **Powerbank**: Compact cube, green LED, USB-C port
5. **Biometric Sensors**: Circular array on bottom, 6-8 LEDs (green/red)

### Connector System
- **Inside Ring**: 4 copper magnetic pins for core device attachment
- **Outside Top Ring**: 6 copper pins for powerbank connection
- **Core Device Bottom**: Biometric sensor array
- **Strap Bottom**: 6 sensor windows with LEDs

### LED States
- **Sensors OFF**: Copper appears silver/gray
- **Sensors ON**: Green LEDs glowing (health monitoring)
- **Powerbank LED**: Green indicator when active
