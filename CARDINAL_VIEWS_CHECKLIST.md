## C4D BATCH RENDERING TIP

Use **Takes Manager** to set up all combinations:
1. Create Take for each component configuration
2. Create Take for each sensor state
3. Create Take for each strap type
4. Add 6 camera positions per combination
5. Add all to Render Queue
6. Batch render overnight

Or render in groups (all FLAT first, then CIRCULAR).
